HTML

HTML stands for ( Full Form ) Hyper Text Markup Language. It is used for Webpage designing and Website Development. HTML documents are written in special codes which are understood by any Web Browsers to do a specific task defined by the user. HTML Tags are codes which defines a specific region in a HTML document to design a specific content of a webpage. HTML Tags are written within <> (angular braces). HTML tags have some properties which can optionally be used in designing an attractive webpage and to change the default appearance according to user requirement, is known as "Attributes". HTML provides a method to format Texts, Create Links, Lists, Insert Images, etc. which is used in a Webpage.
